# GenVoice AI Assessment

## Setting Up

1. **Clone repository**
   ```bash
   git clone https://github.com/yourusername/genvoice-ai-assessment.git
   cd genvoice-ai-assessment
   npm i
   ```

2. **Create environmental variable**
   ```.env
   NEXT_PUBLIC_JWT_SECRET="GenVoiceAI"
   ```

## Usage

#### Running

```bash
npm run dev
```

#### Testing

```bash
npm run test
```
